# Link Title Fixer

Automatically converts pasted URLs into Markdown links with page titles in Obsidian.

This follows the same pattern as Google Docs link paste behavior: when you paste a URL, it fetches the page title and inserts a formatted link instead of leaving a raw URL.

## Quick start

- `cd .obsidian/plugins`
- `git clone git@github.com:jiito/link-title-fixer.git`
